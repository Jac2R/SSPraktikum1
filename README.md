# SimbirSoft-Praktikum1: Задание UI PyTest+Selenium

---

## Описание проекта | Project Description

__

**Чек-лист по заданию:**

1. **Фильтрация в категориях**

    В любой из категорий (не менее 4 товаров) Проверить сортировку товаров по имени и стоимости в
    обоих направлениях - по возрастанию и убыванию

2. **Проверка поисковой выдачи и корзины**

    В поиск ввести "shirt", отсортировать по наименованию и положить в корзины 2 и 3 товар из выдачи
    с рандомным количеством. В корзине: Перейти в корзину, найти самый дешевый товар, удвоить
    его количество. Проверить итоговую сумму.

3. **Проверка корзины**

    Добавить в корзину 5 рандомных товаров с главной страницы с рандомным количеством. В корзине: В
    таблице удалить все четные по порядку товары из корзины, проверить итоговую стоимость

## Как запустить автотесты? | How to run Tests?

### Локальная установка проекта | Local Installation Guide

Перед загрузкой репозитория убедись, что у тебя установлена версия Python 3.10. Инструкция по установке версии 3.10 находится тут.

1. Клонируй репозиторий с проектом в свою любимую папку с проектами:
```
    git clone git@github.com:Jac2R/SSPraktikum1.git
```
2. Установи модули, использовав команду:
```
    pip install -r requirements.txt
```
3. Запусти тесты:
```
    pytest -v
```
4. Запусти Allure report:
```
    pytest --alluredir=allure-results
    allure serve allure-results
```

### Задание 1_TC01 - Сортировка товаров по стоимости - По возрастанию

**Предусловия**

Открыть https://automationteststore.com/

**Шаги**

1. Открыть главную страницу сайта
2. Перейти в категорию Skincare
3. В выпадающем списке сортировки Sort By выбрать "Price Low > High"
4. Считать цены всех отображаемых товаров

**Ожидаемый результат**

Все товары отсортированы по цене: от минимальной к максимальной
